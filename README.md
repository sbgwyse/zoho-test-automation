
# Zoho Test Automation

A Playwright automation framework for testing Zoho Desk APIs and UI.

## 🚀 Features

- UI Automation
- API Automation
- Playwright Framework
- TypeScript
- HTML Reports
- Excel Reports
- PDF Reports

## 📁 Project Structure

```
tests/
pages/
utils/
reporters/
docs/
```

## 🛠 Technologies Used

- Playwright
- TypeScript
- Node.js
- GitHub Actions

## 📚 Zoho Desk API

This project contains automation examples for Zoho Desk APIs.

Official Documentation:
https://desk.zoho.com/DeskAPIDocument
